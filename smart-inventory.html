Project Details

Smart Inventory Management System

A full-stack ASP.NET Core MVC web application for managing products, categories, inventory levels, and orders, with advanced features such as authentication, AJAX interactivity, testing, and Azure deployment.

Project Overview

The Smart Inventory Management System was developed as a two-part academic project focused on building a practical inventory platform for small businesses. The system supports product and category management, stock monitoring, order handling, and a clean, user-friendly interface.

In the second phase, the application was expanded with secure user authentication, role-based authorization, custom user profiles, AJAX-enhanced interactions, stronger error handling, and live deployment to Microsoft Azure.

Technologies Used

ASP.NET Core MVC C# .NET 8 Entity Framework Core PostgreSQL HTML CSS JavaScript AJAX ASP.NET Core Identity Azure Git & GitHub

Core Features

Key functionality implemented across both assignment phases.

Inventory Management

Users can view products, organize items by category, monitor stock levels, and manage inventory records through add, edit, and delete operations.

Search, Filter, and Sort

The application supports product search by name along with filters such as category, price range, and low-stock status, plus sorting options for better usability.

Guest and User Orders

The system supports order placement workflows, beginning with guest order tracking and later expanding to authenticated user ordering with improved account integration.

Authentication and Roles

ASP.NET Core Identity was used to implement registration, login, email verification, password recovery, and role-based restrictions for admins and regular users.

AJAX Interactivity

AJAX was used to improve the user experience by enabling dynamic product search and smoother form submissions without full page reloads.

Error Handling and Deployment

The project includes structured error handling, custom error pages, logging, and deployment to Microsoft Azure to support a more production-ready workflow.

What I Learned

This project strengthened my understanding of full-stack web development using the MVC pattern. I gained experience designing database-backed applications, handling user authentication and authorization, improving front-end usability with AJAX, and working with deployment and testing practices.

Why This Project Matters

The Smart Inventory Management System demonstrates my ability to build practical business-focused software with both technical depth and user-centered design. It reflects experience in backend logic, database design, UI development, and modern web application architecture.

Project Scope Summary

This project began with foundational inventory management, product browsing, filtering, sorting, and guest order handling. It then expanded into a more advanced platform featuring authentication, role-based access control, custom profiles, error handling, AJAX, testing, and cloud deployment. Together, these phases show the progression from a basic CRUD application to a more complete and professional web solution.
