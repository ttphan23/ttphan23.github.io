Project Details

Gomoku AI Game

A command-line Java implementation of the classic strategy game Gomoku (Five in a Row) with support for both human players and an AI opponent powered by the Minimax algorithm.

Project Overview

The Gomoku AI Game is a Java-based command-line application that recreates the classic board game Gomoku on a 9×9 grid. The application allows players to compete against another human player or challenge an AI opponent that evaluates board states using the Minimax algorithm.

The project focuses on implementing game logic, managing board state using arrays, and applying algorithmic decision-making to create an intelligent computer opponent.

Technologies Used

Java Command Line Interface 2D Arrays Object-Oriented Programming Recursion Minimax Algorithm Algorithm Design Git & GitHub

Key Features

Main gameplay and algorithmic features implemented.

Two Gameplay Modes

Players can choose between a two-player mode where two human players compete, or a single-player mode where the user competes against the AI.

9×9 Board System

The game board is implemented using a two-dimensional array that stores the state of each cell and updates after every move.

Win Detection

The game continuously checks for five consecutive pieces horizontally, vertically, or diagonally to determine a winner.

Minimax AI

The AI opponent evaluates possible board states using the Minimax algorithm to determine the optimal move at each turn.

Turn-Based Game Logic

Players take turns placing pieces on the board, with the game updating and validating the board state after every move.

Command-Line Interface

The game runs entirely in the terminal, displaying the board state and prompting users for moves during each turn.

What I Learned

This project strengthened my understanding of algorithmic thinking and game design. Implementing the Minimax algorithm helped demonstrate how recursive algorithms can be used to simulate future game states and choose optimal moves.

Project Significance

The Gomoku AI Game demonstrates my ability to implement algorithms, work with data structures such as arrays, and design interactive command-line applications in Java.

Assignment Context

This project was developed as part of a Data Structures and Algorithms course and required implementing the game using only basic arrays for board representation while developing an AI opponent using the Minimax algorithm.
