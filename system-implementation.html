System Implementation

Final implementation details and demonstration of the capstone project.

Overview

The capstone project was implemented as a mobile application designed for video game discovery, reviews, and personal library management. Users can track games they have played, are currently playing, or plan to play, while also interacting with other users through reviews and ratings.

Technologies Used

  • Frontend: Android (Java)
  • Backend: Node.js, Express
  • Database: MongoDB
  • Authentication: Firebase Authentication

Key Features Implemented

  • User authentication (login/register)
  • Game library management (Played, Playing, Backlog)
  • Reviews and ratings system
  • Feed displaying user reviews
  • Trending games functionality
  • User profile and settings
